# Error / Blocker Log

Use this file to list build errors, blockers, and the current status for each one.

## How to fill it out
- Add one item per issue
- Include the command that failed
- Paste the exact error message
- Note the suspected cause if you know it
- Mark the status as `open`, `investigating`, or `fixed`
